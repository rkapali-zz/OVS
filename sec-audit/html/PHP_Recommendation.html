PHP Recommendation

In case cetain modules aren't installed, you can simply install by the following command sudo apt-get install php5-MODULE_NAME In case they are lower versions than the given standard, follow these steps:

sudo apt-get update

sudo apt-get install --only-upgrade

To search for desired packages following command can bw used. It displays installed version of the package and the next available version.

sudo apt-cache search

sudo apt-get install --only-upgrade
